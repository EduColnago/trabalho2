MyCars, um site de venda de carros online contendo todas as operações CRUD.
Create - Criar um usuário
Read - Ler carros e usuário
Update - atualizar um carro ( não funcionando corretamente)
Delete - Deletar carro ao comprar

Passo a Passo usuário:
Primeiro clicamos em 'Cadastrar-se' para criarmos uma conta, depois 'login' e digitamos nossas credenciais para logarmos no site.
Quando estivermos logado se formos um usuário comum podemos comprar o carro.
Quando clicamos em comprar e confirmamos o carro é deletado do sistema.
Se clicarmos em sair, deslogamos do site.

Passo a Passo admin:
Caso seja um usuário admin você pode editar as informações do carro clicando em 'editar' (não está funcionando corretamente, pois os carros não estão sendo editados)
Se clicarmos em sair, deslogamos do site.

OBS: sistema de fotos dos carros não está funcionando corretamente, por isso botamos uma imagem padrão para todos os carros.

OBS2: tivemos que usar o virtual enviroment e o Pillow (para as imagens).
